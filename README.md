# Learn-Javascript
Learning javascript while building the CounterApp 

JavaScript: - Functions: How to call a block of code when a button is clicked.

Variables: Storing and updating numbers (e.g., let count = 0).

DOM Manipulation: Updating the text on the screen using JavaScript.

📝 Learning Notes & Recap
JavaScript Reminders
Incrementing: count = count + 1 (or count += 1) updates the value stored in the variable.

The Console: Use console.log() to check if functions are firing correctly.

Strings vs Numbers: Remember that 1 + 1 is 2, but "1" + "1" is "11".
